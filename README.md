# 🚖 Bengaluru Ola Executive Dashboard

A professional **Power BI Executive Dashboard** built to analyze **Ola ride booking data** in Bengaluru. This dashboard provides business insights into booking performance, customer behavior, payment methods, vehicle usage, ride status, and pickup/drop locations.

---

## 📊 Dashboard Preview

> **Bengaluru Ola Executive Dashboard**

The dashboard includes:

- 💰 Total Booking Value
- 🚕 Total Rides
- ✅ Successful Rides
- ❌ Incomplete Rides
- ⭐ Customer Ratings
- 💳 Payment Method Analysis
- 🚗 Vehicle Type Distribution
- 📍 Pickup & Drop Location Analysis
- 📅 Date Filter
- 🔍 Interactive Slicers

---

# 📌 Key Metrics

| KPI | Description |
|------|-------------|
| Booking Value | Total revenue generated from bookings |
| Total Rides | Total number of rides |
| Successful Rides | Completed ride count |
| Incomplete Rides | Cancelled/Incomplete ride count |
| Customer Rating | Overall customer satisfaction score |
